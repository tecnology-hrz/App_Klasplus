Instrucciones de Uso

  1. Selecciona la institución para la cual deseas registrar estudiantes
  2. Descarga la plantilla Excel haciendo clic en "Descargar Plantilla Excel"
  3. Completa la plantilla con los datos de los estudiantes
  4. Guarda el archivo y cárgalo usando el botón "Cargar Archivo Excel"
  5. Revisa la vista previa de los datos
  6. Confirma el registro para cargar los estudiantes a la base de datos
